This is just an example, please feel free to add extra fields or change the data representation in any way!

## Json

**Object:**

```json
{
   "Key":"Value"
}
```


**List:**

```json
[
   "Value",
   "Value" 
]
```
**OR**

```json
[
    {
        "Key":"Value"
    },
    {
        "Key":"Value"
    },
]
```

**Note `values` can be objects or lists:**

```json
{
    "Key":{ "Key":"Value" },
    "Key2":[ "Value" ]
}
```
